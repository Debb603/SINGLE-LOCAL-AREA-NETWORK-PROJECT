# SINGLE-LOCAL-AREA-NETWORK-PROJECT

## Overview
This project demonstrates the configuration of a basic single Local Area Network (LAN) using Cisco Packet Tracer. The network was designed to enable communication between connected devices through a switch and server within the same network environment.

## Network Components
- 3 End Devices (PCs)
- 1 Server
- 1 Switch
- Copper Straight-Through Cables

## Objectives
- Configure a basic LAN topology
- Assign IP addresses to connected devices
- Enable communication between devices on the network
- Understand basic network connectivity and device interaction

## Features
- Device-to-device communication
- Server integration within the LAN
- Basic IP addressing configuration
- Connectivity testing using ping commands

## Tools Used
- Cisco Packet Tracer

## Skills Gained
- Network configuration
- IP addressing
- Switch connectivity
- Basic troubleshooting
- Network communication testing

## Project Outcome
Successful communication was established between all connected devices within the network, demonstrating proper LAN setup and configuration.
